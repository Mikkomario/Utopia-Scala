# Utopia Scribe Client - list of changes

## v1.0.6 (in development)
Adds support for **Scribe-Core v1.2** and **Flow v2.7**

## v1.0.5 - 26.05.2025
Adds support for **Access v1.6** & **Disciple v1.9**

## v1.0.4 - 23.01.2025
This update merely reflects some changes introduced in **Flow v2.5.1**.
### Other changes
- Fixed a deprecation warning, reflecting **Flow** changes

## v1.0.3 - 04.10.2024
Built with Scala v2.13.14. No other changes.

## v1.0.2 - 28.07.2024
This update adds support for **Flow v2.4** and **Annex v1.8**
### Other changes
- In some instances where **Vector** was used, **Seq** is now used

## v1.0.1 - 22.01.2024
Supports **Annex v1.7**
### Other changes
- Scala version updated to 2.13.12

## v1.0 - 27.09.2023
Initial release
